Device Function shape
keyboard is an input device used to enter characters and functions into the computer system by pressing buttons.
mouse is an input device that is used with a computer.
Microphone is a device that captures audio by converting sound waves into an electrical signal.
scanner is a device that captures images from photographic prints, posters, magazine pages, and similar sources for computer editing and display.
Barcode reader is an optical scanner that can read printed barcodes, decode the data contained in the barcode and send the data to a computer.
Monitor is a peripheral device which display computer output on a screen
Printer is a peripheral device that produces a physical copy of thev computer’s output
Plotter is a printer that uses a pen that moves over a large revolving sheet of paper and used to print graphic vectors
Projector is an optical device that projects an image (or moving images) onto a surface
Speaker the purpose of speakers is to produce audio output that can be heard by the listener.
